# WebDev

Repository containing the main project and assignments from the Web Development section of MAC0350.

## Main Project: Pokémon Battle Simulator

This project implements a simulator inspired by the traditional battle system from the Pokémon franchise.

The application allows the user to select any Pokémon from the first to fifth generation and simulate a battle against another random Pokémon. The simulator uses moves from the original games and reproduces the basic mechanics of the classic turn-based battle system.

Each battle follows the traditional turn-based format, where players choose moves and the simulator handles damage calculation, status updates, and the progression of the fight until one Pokémon faints.

## Technologies

The project is built using standard web technologies:

- HTML  
- CSS  
- JavaScript
- FastAPI
- SQLModel
